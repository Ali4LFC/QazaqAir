# 🌬️ Air Astana Monitor

Веб-приложение для **мониторинга качества воздуха в реальном времени** в городе Астана, Казахстан.
Данные получаются из внешнего API [WAQI (World Air Quality Index)](https://waqi.info/) и отображаются на интерактивной карте с автоматическим обновлением каждые 30 секунд.

---

## 📋 Описание проекта

Приложение решает задачу отображения актуальных экологических данных без необходимости каждый раз заходить на сторонние сайты. Пользователь открывает одну страницу и видит:

- **Индекс качества воздуха (AQI)** по стандарту US EPA с цветовой индикацией
- **Погодные данные** — температура, влажность, скорость ветра
- **Интерактивную карту** Астаны с маркером на точке наблюдения

## 🏗️ Архитектура

Проект построен по классической схеме **клиент–сервер**:

```
┌─────────────────────────────────────────────┐
│              Браузер (Клиент)               │
│   index.html + style.css + script.js        │
│   Leaflet.js (карта)                        │
│                    │                        │
│   GET /api/air-quality (каждые 30 сек)      │
└────────────────────┬────────────────────────┘
                     │ HTTP
┌────────────────────▼────────────────────────┐
│           Python-сервер (FastAPI)           │
│                 app.py                      │
│   - Хранит Токен в .env (безопасно)         │
│   - Кеширует ответ на 30 сек               │
│   - Трансформирует данные WAQI -> JSON      │
└────────────────────┬────────────────────────┘
                     │ HTTPS
┌────────────────────▼────────────────────────┐
│           WAQI API (waqi.info)              │
│   /feed/Astana/?token=...                   │
└─────────────────────────────────────────────┘
```

**Ключевые архитектурные решения:**

| Решение | Обоснование |
|---|---|
| API-токен хранится только на сервере (`.env`) | Токен никогда не попадает в браузер пользователя |
| Серверное кеширование (30 сек) | Экономия лимита запросов и стабильность работы |
| FastAPI как бэкенд | Асинхронный, быстрый, автодокументация `/docs` |
| Leaflet.js для карт | Бесплатная Open Source библиотека, темная тема через CARTO |

## ⚙️ Технологии

|---|---|
| `FastAPI` | Web-фреймворк, REST API |

|---|---|

## 🚀 Установка и запуск

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Получить API-токен WAQI

1. Зайти на [aqicn.org/api/](https://aqicn.org/api/)
2. Ввести почту и получить токен

### 3. Создать файл `.env`

Создать файл `.env` в корне проекта:

```env
WAQI_TOKEN=ваш_токен_здесь
```

> ⚠️ Никогда не публикуйте `.env` файл в открытый репозиторий!

### 4. Запустить сервер

```bash
py app.py
```

### 5. Открыть в браузере

```
http://127.0.0.1:8000
```

## ⚠️ Известные особенности

- **Лимит IQAir** — бесплатный план даёт 10 000 запросов в месяц. Кеш на 30 сек снижает нагрузку с 86 400 до ~2 880 запросов/сутки
- **Название города** — IQAir хранит Астану с параметром `state=Akmola` (Акмолинская область), а не `state=Astana`
- **HTTP (не HTTPS)** — локальный запуск работает по HTTP; для продакшна потребуется настроить HTTPS через nginx/reverse proxy
