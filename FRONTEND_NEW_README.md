# QazaqAir - Новый Frontend

Модернизированный фронтенд на React + TypeScript + Vite + Tailwind CSS.

## Структура проекта

```
frontend_new/
├── src/
│   ├── api/           # API клиенты
│   │   └── client.ts
│   ├── components/    # UI компоненты
│   │   └── ui/
│   │       ├── Button.tsx
│   │       ├── Card.tsx
│   │       ├── Input.tsx
│   │       ├── Label.tsx
│   │       └── Select.tsx
│   ├── context/       # React Context
│   │   └── AuthContext.tsx
│   ├── lib/           # Утилиты
│   │   └── utils.ts
│   ├── pages/         # Страницы
│   │   ├── Home.tsx      # Главная с картой AQI
│   │   ├── Login.tsx     # Вход
│   │   ├── Register.tsx  # Регистрация
│   │   └── Profile.tsx   # Личный кабинет
│   ├── types/         # TypeScript типы
│   │   └── index.ts
│   ├── App.tsx        # Корневой компонент
│   ├── main.tsx       # Точка входа
│   └── index.css      # Глобальные стили (Tailwind CSS v4)
├── package.json
├── tsconfig.json
├── vite.config.ts
└── index.html
```

## Технологический стек

- **React 19** + **TypeScript**
- **Vite** - сборщик
- **Tailwind CSS v4** - стилизация
- **React Router v7** - роутинг
- **React Hook Form** + **Zod** - формы и валидация
- **Axios** - HTTP клиент
- **React Leaflet** - карты
- **Lucide React** - иконки

## Запуск для разработки

```bash
cd frontend_new
npm install
npm run dev
```

Фронтенд запустится на http://localhost:5173 и будет проксировать API запросы на http://localhost:8000

## Сборка для продакшена

```bash
cd frontend_new
npm run build
```

Сборка будет создана в папке `dist/`. Для интеграции с бэкендом FastAPI:

1. Скопируйте содержимое `dist/` в папку `frontend/` (заменив старые файлы)
2. Или настройте FastAPI на раздачу статики из `dist/`

## Функциональность

### 1. Авторизация и регистрация
- JWT-based аутентификация
- Сохранение токена в localStorage
- Автоматический редирект на логин при 401 ошибке

### 2. Личный кабинет
- Просмотр профиля
- Изменение имени
- **Выбор города** из списка регионов Казахстана
- Сохранение выбранного города в профиле

### 3. Главная страница
- Отображение AQI для выбранного региона
- Интерактивная карта с границами Казахстана
- Погодные данные (температура, влажность, ветер)
- Рейтинги: топ-10 чистых и загрязненных регионов
- Переключение языка (RU/KK)
- Переключение темы (светлая/темная)
- Боковая панель со списком всех регионов

## API интеграция

Бэкенд предоставляет следующие эндпоинты:

### Auth
- `POST /api/auth/register` - регистрация
- `POST /api/auth/login` - вход (OAuth2 формат)
- `GET /api/auth/me` - получить профиль
- `PATCH /api/auth/me` - обновить профиль

### Air Quality
- `GET /api/air-quality` - текущие данные
- `GET /api/air-quality?region={key}` - данные по региону
- `GET /api/regions` - список регионов
- `GET /api/summary` - сводка (топ чистых/грязных)

## Особенности реализации

1. **Двуязычность** - поддержка русского и казахского языков
2. **Темная тема** - автоматическое переключение темы
3. **Адаптивный дизайн** - работает на мобильных устройствах
4. **Карта** - интерактивная карта с границами Казахстана
5. **Выбор города** - при выборе города в профиле, он автоматически используется на главной странице
